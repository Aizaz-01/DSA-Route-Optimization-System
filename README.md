# DSA Route Optimization System

A desktop-based **Route Optimization System** developed using **C++** and **Qt Creator** as a semester project for **Data Structures & Algorithms (DSA)**.

The project demonstrates route optimization and pathfinding concepts using efficient algorithms with an interactive graphical user interface.

---

# Features

* Interactive GUI built with Qt Creator
* Route Optimization Visualization
* Graph-Based Path Representation
* Fast Route Calculation
* User-Friendly Interface
* Hybrid Optimization Technique
* Efficient Data Structures Usage

---

# Technologies Used

* **C++**
* **Qt Creator**
* **Data Structures & Algorithms**
* **Graph Algorithms**

---

# Algorithms Implemented

* Greedy Algorithm
* Brute Force Algorithm
* Hybrid (Greedy+ACO+2OPT) Algorithm

---

# Project Structure

```bash
main.cpp          # Main application source code
.pro              # Qt project configuration file
```

---

# How to Run

## Requirements

* Qt Creator
* C++ Compiler (MinGW/MSVC)

## Steps

1. Clone or download this repository
2. Open the `.pro` file in Qt Creator
3. Configure the compiler kit
4. Click the **Run** button

The objective of this project is to demonstrate how Data Structures and Algorithms can be applied to solve real-world route optimization and pathfinding problems efficiently.

The project focuses on:

* Optimization techniques
* Algorithm efficiency
* Graph traversal concepts
* GUI-based visualization
